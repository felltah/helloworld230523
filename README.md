 + top
      + 시스템의 상태를 빠르게 파악(CPU, Memory, Process)
      + interval 간격(기본 3초)으로 화면을 갱신하며 정보를 보여준다.

```bash

top -b #batch 모드, 순간의 정보
top -n #top 실행 주기 설정(반복 횟수)
```
